##Zetta FONA Phone serial device driver

###Install

```
$> npm install zetta-fona-phone-serial-driver
```

###Usage

```
var zetta = require('zetta');
var FonaPhone = require('zetta-fona-phone-serial-driver');

zetta()
  .use(FonaPhone)
  .listen(1337)
```
